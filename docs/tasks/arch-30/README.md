## 📝 文档模板

每个任务开发指南遵循统一的模板结构：

1. **任务概述** - 目标、范围、依赖
2. **前置条件检查** - 必须满足的条件
3. **文件操作清单** - 新建/修改的文件
4. **实现步骤** - 详细的分步指导
5. **测试指南** - 测试用例和验证方法
6. **Git 提交规范** - 提交信息模板
7. **验收标准** - 完成检查清单
8. **常见问题** - 预防和解决方案
