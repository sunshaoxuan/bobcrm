# PLAN-03: v0.8.0 开发路线图

**目标**: 实现仪表盘引擎 MVP、字段级安全，并解决技术债务（架构重构）。

## 1. 技术债务（高优先级）

### TASK-01: 提取 TemplateService
*   **来源**: 从 v0.7.0 (FIX-04) 结转。
*   **目标**: 将业务逻辑从 `TemplateEndpoints.cs` 移动到 `TemplateService.cs`。
*   **收益**: 提高可测试性和可维护性；为其他领域潜在的模板逻辑重用做准备。

## 2. 执行计划

1.  **第 1 周**: 架构重构 (TASK-01) & 字段级安全后端 (TASK-03)。
2.  **第 2 周**: 字段级安全 UI & 集成。
3.  **第 3 周**: 仪表盘引擎 MVP (TASK-02) - 设计器。
4.  **第 4 周**: 仪表盘引擎 MVP (TASK-02) - 组件 & 查看器。

## 4. 成功标准

*   `TemplateEndpoints.cs` 变薄 (< 200 行)。
*   用户可以定义哪些角色可以看到/编辑敏感字段（例如，“薪资”）。
*   用户可以创建一个包含至少 3 种图表类型的仪表盘。
